# juzibot-website 本地部署

纯静态站点，无需 Node、无需构建。所有 HTML 已提交在仓库中，直接伺服根目录即可。

## 启动（Windows）

双击 `start-server.bat`，浏览器会自动打开 http://localhost:8000

前提：装有 Python（python.org 下载，任意 3.x 版本）。

## 其他方式

在本目录执行任一：

```
python -m http.server 8000
npx serve .          # 如装有 Node
```

## 注意

- 不要运行 `build_pages.py`——脚本已过时，会覆盖手工维护的页面（见 CLAUDE.md）
- 子页面导航/页脚由 `assets/site.js` 运行时注入，必须通过 HTTP 访问（直接双击 html 文件会因 file:// 限制导致部分功能失效）
- 线上 nginx 会把未知路径 301 回首页，本地简易服务器无此行为，属正常差异
